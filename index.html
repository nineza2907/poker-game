ต้องการ: GitHub Personal Access Token (PAT) ที่มีสิทธิ์ gist

🔑
สร้าง GitHub Token

1. ไปที่ github.com → Settings
2. เลือก Developer settings
3. เลือก Personal access tokens → Tokens (classic)
4. คลิก Generate new token (classic)
5. ตั้งชื่อ เช่น 10k-dashboard
6. ติ๊ก scope: gist เท่านั้น
7. คลิก Generate token แล้ว copy ทันที
